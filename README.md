This project builds an automated ETL pipeline using Apache Airflow to fetch daily stock market data from NSE and store it in a PostgreSQL database.

**Approach:**

1. Extracted NIFTY 50 data using NSE API
2. Transformed data into structured formats (nifty, stocks, metadata)
3. Loaded processed data into Postgres tables
4. Scheduled pipeline to run daily using Airflow

**Tech Stack:**
Python, Airflow, PostgreSQL, Pandas
